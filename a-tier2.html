1. Geolocation API (Advanced Use)

The Geolocation API allows you to retrieve a user's real-world location. In Part 1 we touched the basics ‚Äî now let‚Äôs add accuracy and error handling.


// Request current position with options
navigator.geolocation.getCurrentPosition(
  (position) => {
    console.log("Latitude:", position.coords.latitude);
    console.log("Longitude:", position.coords.longitude);
    console.log("Accuracy (meters):", position.coords.accuracy);
  },
  (error) => {
    console.error("Error getting location:", error.message);
  },
  { enableHighAccuracy: true, timeout: 5000, maximumAge: 0 }
);

Pro tip: Always provide a fallback if the user denies permission.

2. Clipboard API (Writing to Clipboard)

In Part 1 we learned how to read from the clipboard. Now let‚Äôs learn how to write text programmatically.


async function copyText() {
  try {
    await navigator.clipboard.writeText("Copied from A-Tier Part 2!");
    alert("Text copied to clipboard.");
  } catch (err) {
    console.error("Failed to copy: ", err);
  }
}

copyText();

Use case: Copying coupon codes, shareable links, or user-generated content.

3. Fullscreen API

The Fullscreen API lets you programmatically toggle fullscreen mode for elements. Great for video players, games, or immersive apps.


const elem = document.documentElement;

function goFullscreen() {
  if (elem.requestFullscreen) {
    elem.requestFullscreen();
  }
}

function exitFullscreen() {
  if (document.exitFullscreen) {
    document.exitFullscreen();
  }
}

4. Vibration API

The Vibration API can make a mobile device vibrate. It‚Äôs mainly useful for games or notifications.


if ("vibrate" in navigator) {
  // Vibrate for 200ms
  navigator.vibrate(200);

  // Complex vibration pattern: [on, off, on, off...]
  navigator.vibrate([100, 50, 200, 50, 300]);
}

Note: Works only on supported mobile devices.

5. Page Visibility API

The Page Visibility API allows you to detect if a page is visible or hidden. Useful for pausing videos or animations when the user switches tabs.


document.addEventListener("visibilitychange", () => {
  if (document.visibilityState === "hidden") {
    console.log("User left the page ‚Äî pause things.");
  } else {
    console.log("User returned ‚Äî resume things.");
  }
});
