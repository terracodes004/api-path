üì¢ Notification API

The Notification API lets web apps display system notifications outside the browser window.


// Request permission
Notification.requestPermission().then(permission => {
  if (permission === "granted") {
    new Notification("Hello!", {
      body: "This is a web notification.",
      icon: "icon.png"
    });
  }
});

Use case: Reminders, chat alerts, background task updates.

üìã Clipboard API

The Clipboard API enables copy, cut, and paste operations.


// Copy to clipboard
navigator.clipboard.writeText("Copied text!").then(() => {
  console.log("Text copied!");
});

// Paste from clipboard
navigator.clipboard.readText().then(text => {
  console.log("Pasted:", text);
});

Use case: Copy buttons, password managers, text editors.

üïò History API

The History API allows manipulation of the browser session history.


// Add a new entry
history.pushState({ page: 1 }, "title 1", "?page=1");

// Navigate back and forward
history.back();   // Go back
history.forward(); // Go forward

// Listen for state changes
window.onpopstate = event => {
  console.log("State:", event.state);
};

Use case: Single Page Applications (SPAs), dynamic navigation.

üñ•Ô∏è Fullscreen API

The Fullscreen API lets you display elements in full-screen mode.


const elem = document.documentElement;

function openFullscreen() {
  if (elem.requestFullscreen) {
    elem.requestFullscreen();
  }
}

function closeFullscreen() {
  if (document.exitFullscreen) {
    document.exitFullscreen();
  }
}
