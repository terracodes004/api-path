Clipboard API

Clipboard lets you copy and paste content programmatically.

Step 1: Copy text

navigator.clipboard.writeText("Hello from Clipboard API!")
  .then(() => console.log("Copied!"))
  .catch(err => console.error("Failed to copy:", err));

Step 2: Paste text

navigator.clipboard.readText()
  .then(text => console.log("Pasted:", text));
üí° Useful for ‚ÄúCopy to Clipboard‚Äù buttons in websites.

Drag & Drop API

Drag & Drop allows users to move elements around with their mouse.

Step 1: Make an element draggable

<div id="dragItem" draggable="true"
     style="width:100px;height:100px;background:red">
  Drag me!
</div>

<div id="dropZone" style="width:200px;height:200px;border:2px dashed black">
  Drop here
</div>

<script>
const dragItem = document.getElementById("dragItem");
const dropZone = document.getElementById("dropZone");

dragItem.addEventListener("dragstart", e => {
  e.dataTransfer.setData("text/plain", "Dragged!");
});

dropZone.addEventListener("dragover", e => {
  e.preventDefault(); // Allow drop
});

dropZone.addEventListener("drop", e => {
  e.preventDefault();
  alert("Dropped: " + e.dataTransfer.getData("text"));
});
</script>
üí° Used in file uploads, custom UI editors, or task boards like Trello.

Web Storage API

LocalStorage and SessionStorage let you save small data directly in the browser.

Step 1: Save data

localStorage.setItem("username", "Kemi");
sessionStorage.setItem("sessionId", "12345");

Step 2: Retrieve data

console.log(localStorage.getItem("username")); // Kemi

Step 3: Remove data

localStorage.removeItem("username");
üí° Perfect for remembering preferences, but don‚Äôt store sensitive data here.
