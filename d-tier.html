Payment Request API

Simplifies online checkout by integrating browser-stored cards and payment methods.

Step 1: Create a payment request

const request = new PaymentRequest(
  [{
    supportedMethods: "basic-card",
    data: { supportedNetworks: ["visa", "mastercard"] }
  }],
  { total: { label: "Total", amount: { currency: "USD", value: "10.00" } } }
);

request.show()
  .then(paymentResponse => {
    console.log(paymentResponse);
    return paymentResponse.complete("success");
  });
üí° Good idea, but adoption is limited because sites prefer their own checkout systems.

Presentation API

Lets you present web content on a second screen (like Chromecast or smart TVs).

Step 1: Connect to a presentation display

const presentationRequest = new PresentationRequest(["https://example.com/presentation"]);

navigator.presentation.defaultRequest = presentationRequest;

presentationRequest.start().then(connection => {
  console.log("Connected to presentation display!");
});
üí° Interesting for business apps and remote presentations, but adoption is minimal.
