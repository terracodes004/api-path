S-Tier JavaScript APIs (Part 2)

Exploring more of the most powerful and game-changing APIs.

1. File System Access API

The File System Access API lets websites read and write files directly on the user‚Äôs device (with permission).


async function saveFile() {
  const handle = await window.showSaveFilePicker({
    suggestedName: "example.txt",
    types: [{ description: "Text Files", accept: { "text/plain": [".txt"] } }]
  });

  const writable = await handle.createWritable();
  await writable.write("Hello from S-Tier Part 2!");
  await writable.close();
}

saveFile();

Note: Only works in Chromium-based browsers.

2. File Handling API

The File Handling API allows installed web apps (via PWA) to become file handlers for certain file types.


// In manifest.json:
// "file_handlers": [{
//   "action": "/open-file",
//   "accept": { "text/plain": [".txt"] }
// }]

// Example listener:
window.launchQueue.setConsumer((launchParams) => {
  for (const file of launchParams.files) {
    console.log("Opened file:", file.name);
  }
});

3. Shape Detection API

The Shape Detection API gives access to text, barcode, and face detection in images.


if ("FaceDetector" in window) {
  const faceDetector = new FaceDetector();
  const img = document.querySelector("img");

  faceDetector.detect(img)
    .then(faces => console.log("Faces detected:", faces))
    .catch(err => console.error("Error:", err));
}

Use case: AR apps, barcode scanners, smart camera apps.

4. WebHID API

The WebHID API allows web apps to communicate with Human Interface Devices (HID) like game controllers, VR equipment, or custom hardware.


const devices = await navigator.hid.requestDevice({ filters: [] });
console.log("Connected devices:", devices);

5. WebBluetooth API

The WebBluetooth API lets your website connect to Bluetooth devices securely.


navigator.bluetooth.requestDevice({
  acceptAllDevices: true
}).then(device => {
  console.log("Connected to:", device.name);
});

Use case: IoT devices, wearables, smart home gadgets.
