DOM API (Document Object Model)

The DOM is the bridge between JavaScript and your webpage. Without it, JS wouldnâ€™t be able to change text, images, or styles.

Step 1: Select Elements

// Select by tag
const heading = document.querySelector("h1");

// Select by id
const box = document.getElementById("myBox");

// Select by class
const buttons = document.querySelectorAll(".btn");

Step 2: Change Content

heading.textContent = "Hello from JavaScript!";
box.innerHTML = "<strong>This is bold text</strong>";

Step 3: Modify Styles

box.style.backgroundColor = "yellow";
box.style.padding = "20px";

Step 4: Create & Add Elements

const newPara = document.createElement("p");
newPara.textContent = "I was added with JavaScript!";
document.body.appendChild(newPara);

Fetch API

The Fetch API lets you make HTTP requests to servers (APIs). Itâ€™s how you get or send data.

Step 1: Basic Fetch

fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(response => response.json())
  .then(data => console.log(data));

Step 2: POST Request

fetch("https://jsonplaceholder.typicode.com/posts", {
  method: "POST",
  body: JSON.stringify({ title: "Hello", body: "World" }),
  headers: { "Content-type": "application/json; charset=UTF-8" }
})
.then(res => res.json())
.then(data => console.log("Created:", data));

Web Storage API

This API gives you localStorage and sessionStorage to save data in the browser.

Step 1: Save Data

localStorage.setItem("username", "Alice");
sessionStorage.setItem("theme", "dark");

Step 2: Get Data

console.log(localStorage.getItem("username")); // Alice
console.log(sessionStorage.getItem("theme")); // dark

Step 3: Remove Data

localStorage.removeItem("username");
sessionStorage.clear();

if(localStorage.getItem("darkMode") === "on") {
  document.body.style.background = "#333";

    localStorage.setItem("darkMode", "off");
    document.body.style.background = "#fff";
    document.body.style.color = "#000";
  } else {
    localStorage.setItem("darkMode", "on");
    document.body.style.background = "#333";
    document.body.style.color = "#fff";
  }
});
</script>

Events API

The Events API lets you respond to user actions like clicks, typing, or scrolling.

Step 1: Add Event Listener

document.querySelector("button").addEventListener("click", () => {
  alert("Button clicked!");
});

Step 2: Handle Input

document.querySelector("input").addEventListener("input", e => {
  console.log("You typed:", e.target.value);
});

Step 3: Keyboard Event

document.addEventListener("keydown", e => {
  if(e.key === "Enter") {
    console.log("Enter was pressed!");
  }
});
