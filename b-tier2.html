B-Tier JavaScript APIs (Part 2)

Solid APIs that are helpful but more situational than S or A tier.

1. Broadcast Channel API

Lets multiple browser tabs or windows of the same origin communicate.


const channel = new BroadcastChannel("demo");
channel.postMessage("Hello from Tab 1!");

channel.onmessage = (e) => {
  console.log("Message received:", e.data);
};

2. Page Visibility API

Detects when a page is visible or hidden (useful for pausing videos or saving resources).


document.addEventListener("visibilitychange", () => {
  if (document.hidden) {
    console.log("Page is hidden. Pause tasks...");
  } else {
    console.log("Page is visible. Resume tasks...");
  }
});

3. Fullscreen API

Allows elements to be displayed in full screen.


function goFullScreen() {
  const elem = document.documentElement;
  if (elem.requestFullscreen) {
    elem.requestFullscreen();
  }
}

4. Clipboard API (Advanced)

More advanced use of the Clipboard API (reading content instead of just writing).


async function pasteText() {
  const text = await navigator.clipboard.readText();
  console.log("Pasted from clipboard:", text);
}

pasteText();

5. CSS Font Loading API

Allows dynamic loading and controlling of fonts with JavaScript.


const font = new FontFace("MyFont", "url(myfont.woff2)");
font.load().then((loadedFont) => {
  document.fonts.add(loadedFont);
  console.log("Font loaded:", loadedFont.family);
});
