C-Tier JavaScript APIs (Part 2)

Interesting but situational APIs ‚Äî useful in edge cases.

1. Battery Status API (Deprecated in some browsers)

Lets you check device battery levels. (Works in some browsers like Firefox, but deprecated in Chrome.)


navigator.getBattery().then(battery => {
  console.log("Battery level:", battery.level * 100 + "%");
  battery.addEventListener("levelchange", () => {
    console.log("Battery updated:", battery.level * 100 + "%");
  });
});

2. Network Information API

Provides info about the user's connection type and speed.


if (navigator.connection) {
  console.log("Effective connection type:", navigator.connection.effectiveType);
  console.log("Downlink speed:", navigator.connection.downlink, "Mbps");
}

3. Screen Orientation API

Lets you detect and lock the orientation of a device screen.


console.log("Current orientation:", screen.orientation.type);

screen.orientation.addEventListener("change", () => {
  console.log("Orientation changed:", screen.orientation.type);
});

4. Gamepad API (extra feature)

Checks connected controllers.


window.addEventListener("gamepadconnected", (e) => {
  console.log("Gamepad connected:", e.gamepad.id);
});

5. Idle Detection API (Experimental)

Detects when a user is away from device. (Not widely supported yet.)


if ("IdleDetector" in window) {
  const detector = new IdleDetector();
  detector.addEventListener("change", () => {
    console.log("User state:", detector.userState);
    console.log("Screen state:", detector.screenState);
  });
  detector.start();
}
