Canvas API

The Canvas API lets you draw shapes, text, and even create games or animations.

Step 1: Set up the canvas

<canvas id="myCanvas" width="300" height="150" style="border:1px solid black;"></canvas>

<script>
const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");
</script>

Step 2: Draw shapes

ctx.fillStyle = "red";
ctx.fillRect(20, 20, 100, 50); // Rectangle

Step 3: Draw text

ctx.font = "20px Arial";
ctx.fillText("Hello Canvas!", 20, 100);

Step 4: Animations

let x = 0;
function animate() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = "blue";
  ctx.fillRect(x, 50, 50, 50);
  x += 2;
  requestAnimationFrame(animate);
}

Web Workers API

Web Workers let you run JavaScript in the background, so your app doesnâ€™t freeze during heavy tasks.

Step 1: Create a worker file

// worker.js
self.onmessage = function(event) {
  let num = event.data;
  let result = 0;
  for (let i = 0; i < num; i++) result += i;
  self.postMessage(result);
};

Step 2: Use the worker

const worker = new Worker("worker.js");
worker.postMessage(1000000);
worker.onmessage = (event) => {
  console.log("Result from worker:", event.data);
};

File API

The File API allows users to upload, read, and process files directly in the browser.

Step 1: File input

<input type="file" id="fileInput">

Step 2: Read file content

const input = document.getElementById("fileInput");
input.addEventListener("change", (event) => {
  const file = event.target.files[0];
  const reader = new FileReader();

  reader.onload = function(e) {
    console.log("File content:", e.target.result);
  };

  reader.readAsText(file);
});
