Obsolete, Experimental, or Barely Useful APIs

1. Battery Status API (Deprecated)

This API once allowed websites to detect your battery percentage and charging status. It was deprecated due to privacy concerns and is now removed from most browsers. 


// Deprecated example
navigator.getBattery?.().then(battery => {
  console.log("Battery level:", battery.level * 100 + "%");
});

2. Network Information API

Provided details about the user‚Äôs network type (wifi, 4g, etc.). Today it‚Äôs only supported in some Chromium-based browsers, and it is very unreliable. 


if ('connection' in navigator) {
  console.log("Effective network type:", navigator.connection.effectiveType);
}

3. Ambient Light Sensor API

Experimental API to detect the ambient light around the device. Could be used to adjust brightness, but has serious privacy risks. 


if ('AmbientLightSensor' in window) {
  try {
    const sensor = new AmbientLightSensor();
    sensor.addEventListener('reading', () => {
      console.log("Ambient light:", sensor.illuminance);
    });
    sensor.start();
  } catch (err) {
    console.error("Sensor error:", err);
  }
}

4. Device Orientation Events (Legacy)

Still around, but modern sites use the Generic Sensor API instead. This legacy version often requires permissions and varies across browsers. 


window.addEventListener("deviceorientation", event => {
  console.log("Alpha:", event.alpha, "Beta:", event.beta, "Gamma:", event.gamma);
});

5. VR Display API (Obsolete)

The old VRDisplay API was replaced by WebXR. Modern VR/AR projects should always use WebXR instead. 


// Obsolete VRDisplay example
if ('getVRDisplays' in navigator) {
  navigator.getVRDisplays().then(displays => {
    console.log("VR Displays:", displays);
  });
}

6. MediaStream Image Capture API

Lets you capture still images from a camera video stream. Rarely used directly ‚Äî most projects use WebRTC or MediaRecorder instead. 


if ('ImageCapture' in window) {
  navigator.mediaDevices.getUserMedia({ video: true }).then(stream => {
    const track = stream.getVideoTracks()[0];
    const imageCapture = new ImageCapture(track);
    imageCapture.takePhoto().then(blob => {
      console.log("Photo captured:", blob);
    });
  });
}
